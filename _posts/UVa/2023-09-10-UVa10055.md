# 題目來源
[UVa 10055](https://onlinejudge.org/index.php?option=com_onlinejudge&Itemid=8&page=show_problem&problem=996)

# 題意

- 給你兩個數字，相減求絕對值

# 思路

- 超級大水題，需要注意幾點
    - 整數存取範圍是$2^{63}$，所以用`long long int`
    - 用EOF的方式輸入
    - 相減求絕對值：用`abs()`
    
    ```cpp
    long long int a,b;
    while (cin>>a>>b){
    	cout<<abs(a-b)<<endl;
    }
    ```
    

# 參考解答

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main()
{
    long long int a,b;
    while (cin>>a>>b){
        cout<<abs(a-b)<<endl;
    }
    return 0;
}
```
